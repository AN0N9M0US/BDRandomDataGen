## Installation

Install my-project with python

1st Run This Command

```bash
  apt install python -y
```
```bash
  apt install git -y
```

Clone the project

```bash
  git clone https://github.com/AN0N9M0US/BDRandomDataGen.git
```

Go to the project directory

```bash
  cd BDRandomDataGen
```

Run

```bash
  python gen.py
```
